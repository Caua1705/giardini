## 📌 Project Overview

Giardini Café is a fully responsive institutional website developed using a Design System–first approach.

The project simulates a professional front-end production workflow, focusing on:

- Visual consistency
- Modular CSS architecture
- Structured project organization
- Responsive behavior across devices
- Controlled animations and performance optimization

The development followed an AI-assisted workflow, where AI was used as a productivity accelerator while architectural decisions and structure design were manually defined.

## 🎯 Project Objectives

- Implement a structured Design System before UI implementation
- Ensure responsive layout across desktop and mobile
- Apply modular CSS architecture
- Maintain visual hierarchy and component reusability
- Simulate real-world production deployment

---

## 🏗 Architecture & Organization

The project follows a structured organization:


assets/ → static resources (images, media)
styles/ → modular CSS structure
scripts/ → JavaScript interactions and animations
design-system → visual references and structural definitions
videos/ → hero and motion assets


### CSS Architecture

CSS is divided into responsibility-based modules:

- `base.css` → resets, typography, global rules
- `layout.css` → structural layout system
- `components.css` → reusable UI components
- `menu.css` → navigation-specific styles
- `animations.css` → motion system and effects

This separation improves scalability and maintainability.
